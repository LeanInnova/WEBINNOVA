## Instalación y Uso

### Requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Servidor web local (opcional para desarrollo)

### Pasos para ejecutar
1. Descarga todos los archivos en una carpeta
2. Abre `index.html` en tu navegador
3. ¡Listo! El sitio web está funcionando

### Para desarrollo local
```bash
# Usando Python
python -m http.server 8000

# Usando Node.js
npx serve .

# Usando PHP
php -S localhost:8000
```

## Personalización

### Imágenes Requeridas
Para completar el sitio web, necesitas proporcionar las siguientes imágenes en la carpeta `assets/`:

1. **logo.png** - Logo de la empresa (recomendado: 200x200px)
2. **hero-image.jpg** - Imagen principal del hero (recomendado: 800x600px)
3. **computer-assembly.jpg** - Imagen de armado de computadoras (recomendado: 600x400px)
4. **about-image.jpg** - Imagen del equipo o instalaciones (recomendado: 600x400px)

### Colores Personalizables
Los colores se pueden modificar fácilmente editando las variables CSS en `styles.css`:

```css
:root {
    --primary-color: #2563eb;    /* Color principal */
    --secondary-color: #64748b;  /* Color secundario */
    --accent-color: #f59e0b;     /* Color de acento */
    /* ... más variables */
}
```

### Información de Contacto
Edita la información de contacto en `index.html`:
- Teléfono
- Email
- Dirección
- Horarios de atención
